# kpiCard
Simple widget for R to show an indicator and its evolution using [htmlwidgets](http://www.htmlwidgets.org/) and [D3.js](https://d3js.org/).

## Install
```
devtools::install_github("mfontcada/kpiCard")
```

## Usage
```
kpiCard::kpiCard("Sales", 125, 100)
```
